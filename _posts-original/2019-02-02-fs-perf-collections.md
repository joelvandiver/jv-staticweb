# PERF Experiments for F# Collection Types

> Define the expected power operator:

```fsharp
open System

let (^) x n = pown x n
let max = Math.Pow(10., 6.)
```

*Output:*
```console
val ( ^ ) : x:int -> n:int -> int
val max : float = 1000000.0
```

## Initialize a Sequence

```fsharp
#time
let aSeq    = { (-1.*max) .. max }
```
